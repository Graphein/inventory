# Inventory Management API

Este projeto é uma API RESTful para gerenciamento de estoque, produtos, matérias-primas e produção, desenvolvida com **Spring Boot**, **Java 17** e **Maven**.

## 📝 Funcionalidades

- CRUD de produtos (`Product`)
- CRUD de matérias-primas (`RawMaterial`)
- Gestão de produção (`Production`)
- Relacionamento de produtos e materiais (`ProductMaterial`)
- Testes unitários e de integração

---

## ⚙️ Tecnologias

- **Java 17**
- **Spring Boot 3.3.x**
- **Maven**
- **JPA / Hibernate**
- **PostgreSQL** (Docker)
- **Lombok** para redução de boilerplate
- **JUnit 5** e **Mockito** para testes
